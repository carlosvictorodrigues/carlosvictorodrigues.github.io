# Ratio — Pesquisa Jurisprudencial com IA

Site oficial do **Ratio**, o primeiro assistente de pesquisa jurisprudencial com IA operando 100% offline.

## Links

- **Site**: [ratiojuris.me](https://ratiojuris.me)
- **Repositório do projeto**: [github.com/carlosvictorodrigues/ratio](https://github.com/carlosvictorodrigues/ratio)
- **Manual de Operação e Pesquisa**: [Abrir no Google Docs](https://docs.google.com/document/d/1rZ2XNo0eouUoUJuacjetvBbXDJZ6jcx8/edit?usp=sharing&ouid=113759300308613331484&rtpof=true&sd=true)
- **Download**: [Google Drive](https://drive.google.com/file/d/16-twvjn8wJ86zj1HCENFoQkdpRt8Aj0F/view?usp=sharing)

## Sobre

O Ratio consulta mais de 1,1 milhão de documentos do acervo jurisprudencial brasileiro (STF, STJ, TJSP) sem acessar a internet e sem inventar respostas. Cada resultado vem ancorado em fontes reais.

## Estrutura do site

- `index.html` — Página principal
- `styles.css` — Estilos
- `app.js` — Lógica do site (downloads, PIX, modal de anúncio)
- `rag-animation.js` — Animação do pipeline RAG na hero section
