Visão Geral do Projeto

O Ratio é uma plataforma avançada de pesquisa jurisprudencial construída com Inteligência Artificial, projetada para rodar localmente no seu computador pessoal.

Diferente de chats comuns que "adivinham" respostas, o Ratio opera sob um princípio rigoroso de Anti-Alucinação: ele não acessa a internet livre para responder.

  • Base de Dados Fixa: Consulta exclusivamente o acervo interno contendo mais de 1.1 milhão de documentos processados.
  • Fluxo RAG: Identifica passagens específicas no acervo e fundamenta a resposta gerando um Dossiê Documental anexo a cada pesquisa.
  • Operação Local: Os arquivos e bancos vetoriais ficam no seu disco (Offline). Apenas a escrita final passa pelo modelo selecionado.
  • Código Aberto: O Ratio é distribuído gratuitamente (Open-Source), garantindo transparência técnica à comunidade jurídica.

Para conhecer a lista completa de funcionalidades (Níveis de Força Normativa, Ferramentas de Áudio, Filtros Judiciais), consulte a documentação oficial ou o GitHub do projeto.
